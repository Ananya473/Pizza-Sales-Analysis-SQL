# Pizza Sales Analysis using SQL

## Project Overview
This project analyzes pizza sales data using SQL to uncover business insights, customer ordering patterns, revenue trends, and product performance.

## Dataset
The dataset contains information about:
- Orders
- Order Details
- Pizza Types
- Pizza Prices and Sizes

## Business Questions Solved

1. Retrieve the total number of orders placed.
2. Calculate the total revenue generated from pizza sales.
3. Identify the highest-priced pizza.
4. Identify the most common pizza size ordered.
5. List the top 5 most ordered pizza types.
6. Calculate the total quantity of each pizza category ordered.
7. Determine the distribution of orders by hour of the day.
8. Analyze category-wise pizza distribution.
9. Calculate the average number of pizzas ordered per day.
10. Determine the top 3 pizza types based on revenue.
11. Analyze cumulative revenue over time and top-performing pizzas by category.

## SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Window Functions
- Date & Time Functions
- Ranking Functions

## Key Insights
- Total Orders: 21,350
- Total Revenue: $817,860.05
- Most Ordered Size: Large (L)
- Highest Priced Pizza: The Greek Pizza
- Best Selling Pizza: The Classic Deluxe Pizza
